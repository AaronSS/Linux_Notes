### 重新启动
如果你一步一步地走到了这里，恭喜你，Arch Linux系统已经安装在你的机器上了。重新启动看看效果吧。

在终端下执行`reboot`命令，即可重新启动系统。然后，如果你看到了一个启动菜单，那就是GRUB的界面，接着系统启动，最后出现一个登陆提示，说明你已经安装成功了！
